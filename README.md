**哈基米航空** 是一个基于 Spring Boot 构建的高并发机票预订系统。系统采用了现代微服务架构中的诸多最佳实践，旨在解决高并发情况下的库存超卖、分布式数据一致性以及性能瓶颈等问题。

## 🚀 核心特性

- **高性能搜票**：使用 Elasticsearch 构建航班索引，结合 Redis 缓存，实现毫秒级的航班搜索与多条件排序（价格、起飞时间等），并具备优雅的数据库降级策略。
- **高并发抢票**：通过 Redis + Lua 脚本实现原子化的库存扣减与底层物理座位占座，彻底杜绝高并发下的“超卖”与“假死座”问题。
- **分布式订单处理**：引入 RabbitMQ 作为消息中间件，通过异步消息解耦订单落库等耗时操作，极大提升接口响应速度。
- **超时取消与容灾恢复**：利用 RabbitMQ 死信队列（DLX）实现订单超时未支付的自动取消。通过天然的 Redis 幂等屏障，确保在系统宕机重试时库存和座位的完美安全回滚，不发生资金纠纷。
- **支付集成**：集成支付宝沙箱支付，具备严密的防重复回调与支付验签机制，通过数据库乐观锁进行最终一致性兜底。
- **安全与鉴权**：使用 JWT 进行无状态的用户身份认证与请求鉴权，敏感配置采用环境变量隔离，保障系统与数据安全。

## 🛠️ 技术栈

- **后端框架**：Spring Boot 3 / Spring Framework
- **持久层框架**：MyBatis-Plus
- **数据库**：MySQL 8.0
- **缓存与分布式锁**：Redis (Lettuce 客户端)
- **消息队列**：RabbitMQ (死信队列、延时重试)
- **搜索引擎**：Elasticsearch
- **安全认证**：JWT, MD5
- **第三方服务**：支付宝沙箱环境 API

## 📂 核心模块说明

- **用户模块 (`UserController`)**：处理用户的注册、登录与 JWT 发放。
- **航班模块 (`FlightController`)**：
  - 提供基于 ES + Redis 的高性能航班搜索 (`search`)。
  - 提供基于数据库查询的降级搜索方案 (`search_flight`)。
  - 核心抢票接口 (`booking`)，集成 Lua 脚本防重、防超卖。
- **订单与异步处理模块 (`BookingAsyncService`, `OrderConsumer`)**：
  - 将海量并发订单的写库操作转为异步 MQ 消费，削峰填谷。
  - 处理订单死信，释放过期资源的库存与座位图。
- **支付模块 (`PayServiceImpl`)**：对接支付宝，生成支付链接与处理支付成功异步回调。

## 💡 架构设计亮点

1. **缓存与数据库的解耦**：Redis 作为高性能前置屏障，阻挡了绝大多数无效请求；MySQL 作为 "Source of Truth" 进行最终一致性兜底。
2. **巧妙的 Lua 脚本原子操作**：在单次 Redis 交互中完成防重校验、扣减宏观库存、锁定微观座位偏移量，性能极高。
3. **奥卡姆剃刀式的状态机设计**：在订单超时取消时，无条件信任 Redis 未支付标记，利用 `srem` 天然防重，大幅简化了状态机异常流转的复杂性。

## 📝 快速开始

1. 克隆本项目：`git clone ...`
2. 配置环境变量：项目启动前需配置好如数据库密码 `${HAKIMI_DB_PWD}`、Redis 密码 `${HAKIMI_REDIS_PWD}`、JWT 密钥 `${HAKIMI_JWT_SECRET}` 等系统环境变量。
3. 中间件准备：确保本地或远程的 MySQL, Redis, RabbitMQ, Elasticsearch 服务已正常启动。
4. 运行 `HakimiAviationApplication` 即可启动服务。
