**哈基米航空**是一个基于 Spring Boot 构建的高并发机票预订系统，旨在解决高并发场景下的库存超卖、分布式数据一致性与性能瓶颈等问题。

## 🚀 核心特性

- **高性能搜票**：基于 Elasticsearch + Redis 二级缓存实现毫秒级航班搜索，具备数据库降级策略。
- **高并发抢票**：通过 Redis + Lua 脚本实现原子化库存扣减与物理座位占位，有效防止高并发下的超卖与重复占座。
- **分布式订单处理**：引入 RabbitMQ 异步解耦订单写库等耗时操作，削峰填谷，提升接口响应速度。
- **超时取消与容灾恢复**：基于 RabbitMQ 死信队列实现订单超时自动取消，结合 Redis 幂等屏障与数据库乐观锁，保障重试场景下的库存与资金一致性。
- **支付集成**：集成支付宝沙箱支付，具备防重复回调与支付验签机制，以数据库乐观锁进行最终一致性兜底。
- **安全与鉴权**：使用 JWT 进行无状态身份认证，敏感配置通过环境变量隔离。

## 🛠️ 技术栈

- **后端框架**：Spring Boot 3 / Spring Framework
- **持久层**：MyBatis-Plus
- **数据库**：MySQL 8.0
- **缓存**：Redis (Lettuce)
- **消息队列**：RabbitMQ（死信队列、延时重试）
- **搜索引擎**：Elasticsearch
- **安全认证**：JWT、MD5
- **第三方服务**：支付宝沙箱 API

## 📂 核心模块

- **用户模块**：注册、登录与 JWT 签发。
- **航班模块**：基于 ES + Redis 的高性能搜索、数据库降级查询、集成 Lua 脚本防重与防超卖的核心抢票接口。
- **订单与异步处理模块**：MQ 消费者异步落库削峰，死信消费者处理超时订单并回滚库存与座位。
- **支付模块**：对接支付宝，生成支付链接，处理支付回调，异步触发退款流程。

## 💡 架构设计亮点

1. **Redis 前置屏障 + MySQL 最终兜底**：Redis 拦截绝大多数无效请求，MySQL 作为数据最终事实来源，职责清晰。
2. **Lua 脚本原子操作**：单次 Redis 交互完成防重校验、库存扣减、座位位图锁定，减少网络往返开销。
3. **幂等状态机设计**：超时取消流程通过 `srem` 原子操作进行幂等判断，结合 `isRedelivered` 区分重复消息与异常重试，简化异常流转处理。
